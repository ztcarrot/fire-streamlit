# 参数说明文档

## 三种预设策略对比

### 1️⃣ 保守策略 (低风险)

适用于谨慎型投资者，假设未来经济环境较差。

**核心特征:**
- 工资增长慢：**2.0%** (未来加薪少)
- 生活开销大：**60%** (消费水平高)
- 存款利率低：**1.5%** (投资回报低)
- 通胀率高：**3.5%** (物价上涨快)

**结果**: 资产积累最慢，退休时总资产最少

---

### 2️⃣ 中性策略 (中等风险)

适用于正常预期，假设未来经济环境平稳。

**核心特征:**
- 工资增长中等：**4.0%** (正常加薪)
- 生活开销中等：**50%** (合理消费)
- 存款利率中等：**2.0%** (正常回报)
- 通胀率中等：**3.0%** (正常物价上涨)

**结果**: 资产积累适中，退休时总资产中等

---

### 3️⃣ 乐观策略 (高增长)

适用于乐观型投资者，假设未来经济环境良好。

**核心特征:**
- 工资增长快：**6.0%** (快速加薪)
- 生活开销小：**40%** (节约消费)
- 存款利率高：**3.0%** (投资回报好)
- 通胀率低：**2.0%** (物价稳定)

**结果**: 资产积累最快，退休时总资产最多

---

## 关键计算规则

### 养老金领取规则
- **缴纳年限**: 养老金需缴纳满 **20年**
- **领取年龄**: **60岁** 后可领取
- **医保年限**: 医保需缴纳满 **25年**
- **提前退休**: 如未满年限需继续缴纳

### 资产计算公式
```
总资产 = 存款 + 公积金 + 个人养老金账户

年度储蓄 = 年收入 + 年养老金 - 年缴纳 - 年生活开销
存款累计 = 上年存款 × (1 + 存款利率) + 年储蓄
```

### 生活开销计算
```
基础开销 = 当地平均工资 × 生活开销比例
月生活开销 = 基础开销 × (1 + 通胀率)^年数
年生活开销 = 月生活开销 × 12
```

## 如何选择策略？

### 适合保守策略的情况:
- 对未来经济预期悲观
- 倾向于高消费、低储蓄
- 担心投资风险
- 希望看到最坏情况下的财务状况

### 适合中性策略的情况:
- 对未来经济预期平稳
- 希望平衡消费和储蓄
- 认为当前环境会持续
- 作为主要的参考基准

### 适合乐观策略的情况:
- 对未来经济预期良好
- 倾向于低消费、高储蓄
- 相信投资会有较好回报
- 希望看到最好情况下的财务潜力

**建议**: 同时运行三种策略进行对比，了解不同情况下的财务状况范围！
